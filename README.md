# 🍽️ Restaurant Bill Calculator

## 📌 Description
A simple Python project that calculates a customer's restaurant bill and applies a 10% discount if the bill is Rs. 5000 or more.

## 🚀 Features
- Takes customer name as input
- Takes bill amount as input
- Applies 10% discount for eligible bills
- Displays original bill, discount, and final bill

## 🛠️ Technologies Used
- Python
- User Input
- Variables
- Arithmetic Operators
- Comparison Operators
- Type Conversion
- If Condition
